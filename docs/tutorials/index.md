Hands-on tutorials for building real applications with toke and ooke. Each tutorial shows the LLM prompts used to generate code, manual build steps, and troubleshooting tips.

## Getting Started

| Tutorial | Difficulty | Time | What you'll build |
|----------|-----------|------|-------------------|
| [Mortgage Calculator (CLI)](/docs/tutorials/mortgage-cli/) | Beginner | ~30 min | CLI app with math, JSON, file I/O |
| [Mortgage Calculator (Web)](/docs/tutorials/mortgage-web/) | Intermediate | ~45 min | Web app with forms, charts, templates |
| [REST API](/docs/tutorials/rest-api/) | Intermediate | ~30 min | JSON API with CRUD operations |
| [Static Site with ooke](/docs/tutorials/static-site/) | Beginner | ~20 min | Blog with markdown, templates, build |

| [Data Pipeline](/docs/tutorials/data-pipeline/) | Intermediate | ~35 min | CSV processing with stats and JSON output |

## How These Tutorials Work

Each tutorial follows the same structure:

1. **What we're building** — feature list and architecture
2. **LLM prompts** — the exact prompts used to generate the toke code
3. **Generated code** — the `.tk` files with annotations
4. **Build and run** — compile commands and expected output
5. **Troubleshooting** — common errors and how to fix them
6. **Exercises** — extend the project on your own

## Prerequisites

- toke compiler installed ([build guide](/docs/compiler/build-deps/))
- A text editor
- Basic familiarity with toke syntax ([Learn toke](/docs/learn/overview/))
