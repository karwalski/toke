Welcome to the toke training course. This is a structured, hands-on introduction to the toke programming language -- a statically typed, machine-native language designed for LLM code generation.

## Who is this for?

Anyone with basic programming experience in any language. You do not need systems programming experience. If you can write a function, a loop, and an if-statement in any language, you are ready.

## What you will learn

By the end of this course you will be able to:

- Read and write toke programs fluently
- Understand why toke exists and where it fits in the LLM ecosystem
- Use the standard library for files, strings, JSON, HTTP, and more
- Handle errors idiomatically using toke's explicit error model
- Structure multi-file projects with modules and imports
- Build a complete CLI application from scratch

## How to use this course

Each lesson includes:

- **Explanation** of concepts with annotated code examples
- **Exercises** to practice what you learned
- **Key takeaways** summarising the important points

Work through the lessons in order. Each one builds on the previous. Type out the examples yourself rather than copying them -- muscle memory matters even for a machine-first language.

## Prerequisites

- A text editor
- The `tkc` compiler installed ([installation guide](/docs/learn/hello-world/))
- Basic programming experience in any language
