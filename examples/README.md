# toke Examples

Working example applications demonstrating toke language features.

| Example | Description | Tutorial |
|---------|-------------|----------|
| [mortgage/](mortgage/) | CLI mortgage calculator with amortisation | [Tutorial](/docs/tutorials/mortgage-cli/) |
| [mortgage-web/](mortgage-web/) | Web app mortgage calculator with charts | [Tutorial](/docs/tutorials/mortgage-web/) |
| [bookmarks-api/](bookmarks-api/) | REST API with JSON CRUD operations | [Tutorial](/docs/tutorials/rest-api/) |
| [tkgrep/](tkgrep/) | CLI file search tool | [Tutorial](/docs/tutorials/cli-tool/) |
| [datapipe/](datapipe/) | CSV data processing pipeline | [Tutorial](/docs/tutorials/data-pipeline/) |

## Building

All examples use the same build pattern:
```bash
tkc <files...> -o <output>
```

See each example's README for specific build commands.

## Prerequisites

- toke compiler (`tkc`) installed
- See [Build Dependencies](/docs/compiler/build-deps/) for platform-specific setup
